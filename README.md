## 📊 무매 4.0 전략

### 기본 원리

- **40분할**: 시드를 40회로 분할 매수
- **별지점**: 평단 기반 매수/매도 가격
- **별% 계산**: `20 - 1.0 × T`
  - T=0: +20% (익절)
  - T=20: 0% (평단)
  - T=40: -20% (손절)

### 매수 조건

```
현재가 < 별지점
```

### 매도 조건

```
현재가 > 별지점 AND
현재가 > 평단 × 1.15 (15% 익절)
```
